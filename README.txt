Customize the way your pagination looks on each post and page with the easy to use interface

== Description ==

Create beautiful, multi-page posts and pages easily with Mediavine Easy Pagination. Never lose your readers halfway through a post, with large customizable buttons that are proven to lead your readers onto the next page


* Easy to use interface makes it simple to adjust your settings
* Integrates seamlessly with Wordpress's built-in Pagination - No need to go back and rewrite content!
* Increase page views, decrease bounce rate, and maximize your ad revenue!


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mediavine-pagination` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Mediavine Pagination Settings screen to configure the plugin
